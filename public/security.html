Sightlines Security

Sightlines is currently a static, browser-based React application deployed on Cloudflare Workers static assets. The v1 app has no server-side application database, no user accounts, and no hosted artwork image storage.

Security Posture

  • Production is served over HTTPS.
  • Security headers are configured through Cloudflare static asset headers.
  • Project data and uploaded artwork images are stored locally in browser storage in v1.
  • No executable installer or desktop binary is distributed from this domain.
  • No third-party tracking scripts are included in the current app shell.

Reporting

Security contact information is published at /.well-known/security.txt. If email routing is not yet available for the domain, use the contact URL listed there.
