Sightlines IT Access Notes

Sightlines is professional productivity software for exhibition, gallery, museum, and art-installation planning. It is not a file-sharing service, social network, advertising network, gambling site, or executable download portal.

Allowlist

  • Hostname: sightlines.art
  • Protocol: HTTPS
  • Port: 443

Current App Behavior

  • Static web application served from Cloudflare Workers static assets.
  • No user account or login required in v1.
  • No hosted project database in v1.
  • No hosted user image storage in v1.
  • No executable downloads.

Suggested Category

Business, productivity, design tool, professional services, or arts/culture business software.
