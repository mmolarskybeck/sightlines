Dropbox PKCE spike — serverless refresh feasibility

Goal: prove in real Safari and Chrome that a pure browser app gets an offline refresh token via PKCE (no secret) and can refresh, upload, list, and delete via CORS.

0. Setup

Create an app at dropbox.com/developers/apps: Scoped access → App folder. Permissions tab: enable files.content.write, files.metadata.read, account_info.read (then re-authorize if you change them). Settings tab → OAuth2 redirect URIs: add exactly . Paste the App key below.
