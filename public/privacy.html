Sightlines Privacy

Sightlines is designed as a local-first planning tool. In the current v1 product, project files and uploaded artwork images are stored in the user's browser storage on their device.

Current Data Handling

  • No account is required.
  • No hosted project database is used.
  • No hosted artwork image storage is used.
  • No payment, advertising, or third-party tracking scripts are part of the app shell.
  • Users should export backup files for long-term storage or sharing.

Network Behavior

The production app is served over HTTPS from sightlines.art. Core editing features are intended to run in the browser after the static application files load.

Future Features

If Sightlines later adds accounts, hosted sync, cloud collaboration, analytics, or payment features, this page should be updated before those features are released.
